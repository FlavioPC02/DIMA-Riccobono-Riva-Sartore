\subsection{Requirements}

\begin{center}
\begin{tabular}{ |c|p{0.8\linewidth}| } 
 \hline
 R1 & The system shall allow the user to register/log-in to the platform.\\
 \hline
 R2 & The system shall allow the user to search for a list of hiking trails.\\
 \hline
 R3 & The system shall allow the user to filter hiking trails based on provided location, difficulty, altimetry, etc. \\
 \hline
 R4 & The system shall allow the user to view a list of hiking trails with all relevant information (e.g. distance, time, elevation, weather condition, difficulty).\\
 \hline
 R5 & The system shall allow the user to save an hiking trail for later use. \\
 \hline
 R6 & The system shall allow the user to download hiking trail essential data. \\
 \hline
 R7 & The system shall allow the user to start an hiking trail with live tracking. \\
 \hline
 R8 & The system shall allow the user to view real time information when in live tracking mode (e.g distance covered, time spent, current elevation). \\
 \hline
 R9 & The system shall allow the user to view real time information in the smartwatch companion app. \\
 \hline
 R10 & The system shall allow the user to pause the current hiking trail. \\
 \hline
 R11 & The system shall allow the user end the current hiking trail. \\
 \hline
 R12 & The system shall allow the user to review, leave a comment and attach photos on the finished hiking trail. \\
 \hline
 R13 & The system shall allow the user end the current hiking trail. \\
 \hline
\end{tabular}
\end{center}
